# 启信宝企业查询 Skill - 用户指南

欢迎使用启信宝企业信息查询 Skill！本指南将帮助你快速开始使用。

## 第一步：获取 API Token

在使用本 Skill 前，你需要获取启信宝 API Token。

**如何获取：**
- 联系启信宝平台管理员获取 API Token
- Token 格式类似：`ent_xxx_xxxxxxxxxx`

## 第二步：配置 Token

### 推荐方式：设置环境变量

这是最安全、最方便的方式，配置一次后永久有效。

#### Windows 用户

1. 按 `Win + R`，输入 `sysdm.cpl`，按回车
2. 点击"高级" → "环境变量"
3. 在"用户变量"中点击"新建"
4. 变量名：`QXBENT_API_TOKEN`
5. 变量值：粘贴你的 Token
6. 点击"确定"保存
7. **重启** Claude、OpenClaw 等 AI 应用

#### Mac 用户

打开终端，编辑配置文件：

```bash
# 如果使用 bash
echo 'export QXBENT_API_TOKEN="your_token_here"' >> ~/.bashrc
source ~/.bashrc

# 如果使用 zsh (Mac 默认)
echo 'export QXBENT_API_TOKEN="your_token_here"' >> ~/.zshrc
source ~/.zshrc
```

#### Linux 用户

打开终端，编辑配置文件：

```bash
echo 'export QXBENT_API_TOKEN="your_token_here"' >> ~/.bashrc
source ~/.bashrc
```

### 临时方式：告诉 AI

如果你使用 Claude、OpenClaw 等 AI 助手，也可以直接在对话中告诉 AI：

```
我的启信宝 API Token 是 ent_xxx_xxx，请记住并在使用 qxbent 时自动使用这个 token
```

**注意**：这种方式仅在当前对话有效，下次对话需要重新告知。

## 第三步：开始使用

配置完成后，你就可以开始查询企业信息了！

### 基本查询示例

**查询企业工商信息：**
```
查询上海合合信息科技发展有限公司的工商信息
```

AI 会返回：
- 企业名称
- 统一社会信用代码
- 法定代表人
- 注册资本
- 成立日期
- 经营状态
- 所属行业
- 注册地址
- 经营范围
- 等等...

**查询企业股东：**
```
帮我查一下上海合合信息科技发展有限公司的股东信息
```

AI 会返回：
- 股东名称
- 持股比例
- 认缴出资额
- 认缴出资日期
- 等等...

**查询主要人员：**
```
查询上海合合信息科技发展有限公司的主要人员
```

AI 会返回：
- 人员姓名
- 职务
- 持股比例
- 等等...

**查询变更记录：**
```
查看上海合合信息科技发展有限公司最近的变更记录
```

AI 会返回最近的工商变更记录。

### 1. 企业名称不完整怎么办？

如果你只知道企业简称，AI 会自动帮你找到候选企业：

```

```

AI 会回复：
```
找到多个匹配的企业，请选择：
1. 胜宏科技（惠州）股份有限公司
2. 惠州市胜宏科技研究院有限公司
3. 南通胜宏科技有限公司
4. 武汉市胜宏科技有限公司
5. 北京胜宏科技有限公司
```

你只需要回复：
```
第一个
```

AI 会自动用完整名称重新查询。

查询上海合合信息科技发展有限公司的工商信息、股东信息和主要人员

## 常见问题

### Q1: 提示"请提供 API Token"？

**A:** 你还没有配置 Token，请按照"第二步"的说明配置。

### Q2: 提示"API Token 无效"？

**A:** Token 可能错误或已过期，请检查：
1. Token 是否完整复制
2. 是否有多余的空格
3. Token 是否已过期（联系管理员）

### Q3: 查询速度慢？

**A:** 这是正常的，因为需要调用启信宝 API。通常查询时间在 1-3 秒。

### Q4: 查询次数有限制吗？

**A:** 是的，API Token 有调用次数限制，具体限制取决于你的 Token 权限。

### Q5: 可以查询哪些企业？

**A:** 可以查询中国境内所有在工商局注册的企业。

### Q6: 数据更新频率？

**A:** 数据来源于官方工商公示系统，更新频率与官方同步。

## 隐私和安全

- Token 仅存储在你的本地环境变量中
- 不会上传到任何服务器
- 查询记录不会被保存
- 请妥善保管你的 Token，不要泄露给他人
