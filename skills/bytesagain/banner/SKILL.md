# Banner
Make big text. Create ASCII banners. Add style to your terminal.
## Commands
- `text <message>` — Generate large text banner
- `box <message>` — Text in a decorative box
- `line [char] [width]` — Decorative line separator
- `header <title>` — Section header with lines
- `styles` — Show available text styles
## Usage Examples
```bash
banner text "Hello World"
banner box "Important Notice"
banner header "Section Title"
banner line = 60
```
